## Installation

_Below are the Required Steps for you to install "Termoji" Locally._

1. Clone the repo
   ```sh
   git clone https://github.com/EmojiPati/Termoji.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run The App.
   ```js
   npm run start
   ```
